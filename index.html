Selected Publications & Research 代表性论文与研究

First Author | IEEE TAFFC (Published)
Phy-FusionNet: A Memory-Augmented Transformer for Multimodal Emotion Recognition
Wu, T., Purwanto, E., & Huang, Y. (2025) DOI: 10.1109/TAFFC.2025.3609046
Addressed the limitation of capturing subtle emotional cues by designing a memory stream module to preserve long-term context. Incorporated a Fourier-based periodicity modeling module for physiological rhythms. The model achieved 13.75% improvement over standard transformers.
针对捕捉细微情感线索的难题，创新性设计了 memory stream（记忆流）模块以保留长时上下文。结合傅里叶周期性建模处理生理节律，该模型在WESAD和CL-Drive数据集上较标准Transformer提升了 13.75% 的性能。
Memory Stream Fourier Analysis Transformer
